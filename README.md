# projeto-back-end

O Projeto Back-End é um projeto desenvolvido como parte de um projeto prático do curso de Desenvolvimento Web Full Stack, promovido pela Geração Tech da IEL Ceará. Esse projeto foi criado com o objetivo de demonstrar, na prática, os conhecimentos adquiridos ao longo das aulas, especialmente na área de Baack-End. Durante o desenvolvimento, aplicamos conceitos fundamentais como  criação de APIs, integração com bancos de dados e autenticação de usuários. A experiência proporcionou uma vivência real de como funciona o processo de criação de uma interface web, desde o planejamento até a entrega final, consolidando o aprendizado de forma prática e colaborativa.
